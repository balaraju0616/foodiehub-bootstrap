# FoodieHub - Bootstrap 5 Project

## Project Overview

This is a simple food delivery website that I created using Bootstrap 5. It has three pages: Home, About, and Contact.

## Steps I Followed

First, I created the basic file structure with three HTML files.
Then I connected Bootstrap using CDN.
After that, I designed each page step by step using components like navbar, cards, and forms.
I used Bootstrap grid system to make the website responsive.
Finally, I added a small JavaScript to make the contact form interactive and deployed the project using GitHub Pages.

## Technologies Used

- HTML
- CSS
- Bootstrap 5
- JavaScript

## Tools Used
